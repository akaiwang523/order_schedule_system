# 訂單與排程管理系統 - 操作說明

## 系統概述

本系統是一個完整的洗衣訂單與排程管理平台，支持三個角色：
- **客戶（Customer）**：下單、查看訂單狀態、上傳照片
- **管理員（Admin）**：管理訂單、更新進度、查看統計
- **員工（Staff）**：查看當日排程（只讀）

## 登入與註冊

### 首次使用
1. 訪問應用首頁
2. 點擊「註冊會員」按鈕
3. 填寫帳號（郵箱）和密碼
4. 提交後自動跳轉到個人資料填寫頁面
5. 填寫姓名、電話、住址並保存

### 登入
1. 在登入頁面輸入帳號和密碼
2. 點擊「登入」按鈕
3. 根據角色自動跳轉到相應首頁

## 客戶功能

### 首頁 - 訂單管理
- **進行中的訂單**：顯示所有未完成的訂單
  - 訂單編號（格式：YYMMDD-序號，如 260426-01）
  - 訂單內容（袋數和送件方式）
  - 下單日期
  - 當前進度（彩色區分）
  - 預計完成日期（如有）

- **已完成訂單**：顯示所有已完成的訂單
  - 點擊訂單編號可查看衣物概況

### 新增訂單
1. 點擊側邊欄「新增訂單」
2. **一、會員資料**
   - 勾選「同會員註冊資料」自動帶入
   - 或手動修改姓名、電話、住址
3. **二、訂單資訊**
   - 選擇送件方式（到府收送/自行送件）
   - 輸入袋數
   - 選擇衣物放置地點（大樓大廳/家門口/其他）
   - 上傳訂單照片（可選）
   - 輸入備註（可選）
4. 點擊「提交訂單」
5. 訂單完成後會顯示成功提示，並停留在原頁面

### 衣物概況
- 當訂單狀態為「已收件」或之後時，點擊「衣物概況」按鈕
- 查看管理員記錄的衣物編號和照片（唯讀）
- 點擊照片可放大預覽

### 個人資料
- 查看和修改個人資訊
- 修改後自動保存

## 管理員功能

### 首頁 - 待處理訂單
- **搜尋功能**：支持按訂單編號、客戶姓名或電話搜尋
- **訂單列表**：顯示所有待處理訂單
  - 訂單編號
  - 客戶姓名
  - 聯絡電話
  - 訂單內容（袋數和送件方式）
  - 當前進度

- **進度更新**
  1. 點擊訂單編號進入詳情
  2. 查看訂單照片（如有）
  3. 選擇進度：已收件 → 清洗中 → 準備送回 → 完成
  4. 如果選擇「完成」，會彈出確認對話框
  5. 確認後訂單從首頁消失

- **當日排程**（如配置）
  - 到府收送：顯示需要上門收送的訂單
  - 自行送件：顯示客戶自行送件的訂單

### 訂單概況
- 按日期篩選查看所有訂單（包括已完成和未完成）
- 支持搜尋功能

### 會員資料
- 搜尋會員
- 查看會員的歷史訂單
- 點擊訂單編號查看詳情

### 營業概況
- 按月份查看營業統計
- 顯示該月份的訂單數量和營業額

## 訂單詳情頁面

### 管理員訂單詳情（/admin/order/:id）
- 顯示訂單基本信息
- 顯示衣物清單
  - 衣物編號（格式：YYMMDD-序號-項目號，如 260426-01-01）
  - 衣物備註
  - 衣物照片
- 功能按鈕
  - 拍照：添加衣物照片
  - 編輯：修改衣物備註
  - 刪除：刪除衣物記錄

### 客戶衣物概況（/customer/order/:id/overview）
- 顯示衣物清單（唯讀）
- 顯示管理員記錄的衣物編號和照片
- 點擊照片可放大預覽

## 進度狀態說明

| 狀態 | 顏色 | 說明 |
|------|------|------|
| 尚未收件 | 灰色 | 訂單已提交，待收件 |
| 已收件 | 藍色 | 已收到衣物 |
| 清洗中 | 黃色 | 正在清洗 |
| 準備送回 | 橙色 | 清洗完成，準備送回 |
| 完成 | 綠色 | 訂單已完成 |

## 常見操作

### 客戶下單流程
1. 登入系統
2. 點擊「新增訂單」
3. 填寫訂單信息
4. 上傳照片（可選）
5. 提交訂單
6. 在首頁查看訂單狀態

### 管理員處理訂單流程
1. 登入系統
2. 在首頁查看待處理訂單
3. 點擊訂單編號查看詳情
4. 拍照記錄衣物信息
5. 更新進度
6. 完成訂單

### 查看訂單歷史
- **客戶**：點擊側邊欄「歷史訂單」
- **管理員**：進入「訂單概況」頁面

## 技術支持

如遇到任何問題，請檢查以下事項：
1. 確保已正確登入
2. 檢查網絡連接
3. 清除瀏覽器緩存
4. 嘗試重新登入

## 系統要求

- 現代網頁瀏覽器（Chrome、Firefox、Safari、Edge）
- 穩定的網絡連接
- 支持 JavaScript 和 Cookies

## 數據安全

- 所有密碼都經過加密存儲
- 個人信息受到保護
- 訂單數據實時同步
